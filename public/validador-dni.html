Validador de DNI / NIE

¿Cómo saber si un DNI es válido?

El sistema de validación español utiliza un algoritmo basado en el Módulo 23. A cada número de 8 cifras le corresponde una letra específica mediante una operación matemática de división.

📄 DNI (Españoles)

Consta de 8 números y una letra de control. Si tienes menos de 8 números, el sistema añade ceros a la izquierda automáticamente.

🌍 NIE (Extranjeros)

Empieza por X, Y o Z. Para el cálculo, la X se sustituye por 0, la Y por 1 y la Z por 2, siguiendo luego el mismo proceso que el DNI.

🔒 Privacidad y Seguridad

En Calculenow nos tomamos en serio tu privacidad. El número introducido no se envía a ningún servidor ni queda registrado en ninguna base de datos. El cálculo se realiza íntegramente en tu dispositivo.

Preguntas frecuentes sobre DNI y NIE

¿Cómo se calcula la letra del DNI?

Se toma el número completo (8 cifras) y se divide entre 23. El resto de esa división (un número entre 0 y 22) se corresponde con una letra según una tabla oficial (donde 0 es T, 1 es R, 2 es W, etc.).

¿Es seguro introducir mi número aquí?

Totalmente. El código de validación se ejecuta en JavaScript local. Esto significa que tu navegador hace la comprobación sin enviar el dato a través de internet. Nosotros no vemos, ni guardamos, ni procesamos tu número de documento.

¿Por qué mi NIE no valida?

Asegúrate de que la letra inicial (X, Y, Z) esté en mayúsculas y que el documento tenga exactamente 9 caracteres. Recuerda que el NIE debe seguir el formato: una letra, siete números y una letra de control final.

¿Sirve este validador para el CIF de empresas?

No, esta herramienta es específica para personas físicas (DNI y NIE). El algoritmo para el CIF (Código de Identificación Fiscal) de empresas es distinto y sigue una lógica diferente.
