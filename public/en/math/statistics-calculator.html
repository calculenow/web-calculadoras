What are mean, median, mode and range?

They are the four basic measures of descriptive statistics. Each one describes the data set from a different angle and together they give a complete picture of how the values are distributed.

📊 Arithmetic mean

The average. Calculated by summing all values and dividing by the count. The most commonly used measure but sensitive to extreme values.

📍 Median

The central value when data are sorted. If there is an even number of values, it is the mean of the two central ones. Not affected by extreme values.

🔁 Mode

The value that appears most often. There can be more than one mode, or none if all values are different.

↔️ Range

The difference between the maximum and minimum value. Measures the total spread of the data set.

Formulas

Mean = Σx / n

Where Σx is the sum of all values and n is the count of data points.

Range = Maximum value − Minimum value

Frequently asked questions about basic statistics

What is the difference between mean, median and mode?

The mean is the arithmetic average. The median is the central value when data are sorted. The mode is the most frequent value. Each one describes the data set from a different angle.

When is it better to use the median instead of the mean?

When there are extreme values that distort the mean. With salaries, for example, a few very high wages raise the mean but the median better reflects the typical salary.

Can there be more than one mode?

Yes. If two or more values appear the same number of times and more than the rest, all of them are modes. This is called a bimodal or multimodal distribution.

What is the range in statistics?

The difference between the maximum and minimum values. It indicates the total spread: the larger the range, the more dispersed the data.
