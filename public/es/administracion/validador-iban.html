¿Qué es el IBAN y cómo funciona?

El IBAN (International Bank Account Number) es un estándar internacional para identificar cuentas bancarias, definido por la norma ISO 13616. Permite realizar transferencias entre países de forma estandarizada y con menor riesgo de errores.

¿Cómo se valida un IBAN?

El algoritmo de validación MOD-97 funciona así:

  1. Se mueven los 4 primeros caracteres al final del IBAN.
  2. Se convierten todas las letras a números (A=10, B=11, ... Z=35).
  3. Se divide el número resultante entre 97.
  4. Si el resto es 1, el IBAN es válido.

Preguntas frecuentes sobre el IBAN

¿Qué es el IBAN?

El IBAN es un código estándar internacional que identifica de forma única una cuenta bancaria. Está compuesto por el código de país, dos dígitos de control y el número de cuenta nacional.

¿Cuántos dígitos tiene un IBAN español?

24 caracteres: ES + 2 dígitos de control + 20 dígitos de cuenta (4 banco + 4 sucursal + 2 control interno + 10 número de cuenta).

¿Cómo se valida un IBAN?

Mediante el algoritmo MOD-97: se reordena el IBAN, se convierten las letras a números y se comprueba que el resultado módulo 97 sea igual a 1.

¿El IBAN es lo mismo que el número de cuenta?

No. El número de cuenta es la parte nacional. El IBAN añade el código de país y los dígitos de control para permitir transferencias internacionales estandarizadas.
