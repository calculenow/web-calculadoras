¿Cómo calcular el IVA en España en 2026?

Calcular el IVA es un proceso esencial tanto para autónomos como para consumidores. El Impuesto sobre el Valor Añadido es un impuesto indirecto que recae sobre el consumo de bienes y servicios.

Tipos de IVA vigentes

  • Tipo General (21%): Aplicable por defecto a la mayoría de operaciones comerciales (ropa, electrónica, servicios profesionales).
  • Tipo Reducido (10%): Se aplica a alimentos para consumo humano, agua, productos farmacéuticos y servicios de hostelería.
  • Tipo Superreducido (4%): Reservado para bienes de primera necesidad (pan, leche, huevos, frutas) y libros o periódicos.

Fórmulas para desglosar y añadir IVA

Para trabajar con el IVA manualmente, puedes utilizar las siguientes fórmulas matemáticas:

Añadir IVA: Importe Total = Base Imponible × (1 + Tipo IVA/100)
Desglosar IVA: Base Imponible = Importe con IVA / (1 + Tipo IVA/100)

Preguntas frecuentes sobre el IVA en España

¿Cuál es la diferencia entre sumar e incluir el IVA?

Sumar IVA se hace sobre el precio neto (base imponible) para hallar el total. Extraer el IVA se realiza sobre el precio final (PVP) para saber cuánto de ese dinero corresponde al impuesto y cuánto al valor real del producto.

¿Qué tipos de IVA existen en España?

Actualmente existen tres tipos:

  • General (21%): Se aplica a la mayoría de productos y servicios.
  • Reducido (10%): Alimentos en general, transporte de viajeros, hostelería y viviendas.
  • Superreducido (4%): Productos de primera necesidad como pan, leche, frutas, libros y medicamentos.
¿Cómo se calcula el IVA manualmente?

Para sumar el IVA al 21%, multiplica la base por 1.21. Para extraerlo de un precio final, divide el total entre 1.21. La diferencia entre el total y ese resultado será la cuota del IVA.

¿Qué es el recargo de equivalencia?

Es un régimen especial de IVA obligatorio para comerciantes minoristas que no transforman los productos que venden. Consiste en pagar un IVA algo más alto a cambio de no tener que hacer declaraciones trimestrales a Hacienda.
