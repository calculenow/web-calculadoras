Preguntas frecuentes sobre área y perímetro

¿Qué diferencia hay entre área y perímetro?

El perímetro es la longitud total del contorno (metros, cm...). El área es la superficie que ocupa la figura (metros cuadrados, cm²...).

¿Cuál es la fórmula del área de un círculo?

Área = π × r². El perímetro (circunferencia) es 2 × π × r.

¿Cómo se calcula el área de un triángulo?

Área = (base × altura) / 2. La altura debe ser perpendicular a la base.

¿Cómo se calcula el área de un trapecio?

Área = ((base mayor + base menor) × altura) / 2.
